# Core Level 1 Ultrasound — Quick‑Stop Guide

## Go live FREE with GitHub Pages (5 steps)
1) Create a GitHub account → New repository → name it `accs-ultrasound-guide`.
2) Upload all files (drag-and-drop).
3) Settings → Pages → Build and deployment: **Deploy from a branch**.
4) Branch: `main`, Folder: `/root` → Save.
5) After ~30–60s, your public URL appears on this page.

## Update content
- Edit `index.html` in GitHub (pencil icon) → Commit changes.
- Use the **Print / Save as PDF** button on the site for a clean handout.

## Add a YouTube video
- Replace the example iframe in the **Video demos** section with your own (YouTube → Share → Embed).
